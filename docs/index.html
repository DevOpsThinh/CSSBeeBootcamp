    base-heading-sizes-calc

    @function base-heading-sizes-calc($heading: 2, $font-family) { ... }
    View source

    Description

    Abstract out the base heading-size into a function that will dynamically generate it based on our base font sizes calculated.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $heading

    The specified level of heading in our page layout with given a default value of 2.

    		Number2
    $font-family

    The name of a font family in our page layout.

    		String or List— none

    Returns

    The level of heading —

    A calculated css-ready heading size values (h1 into h6).

    Throws

    • The first parameter of base-heading-sizes-calc() function must be a number, #{type-of($heading)} was given.

    • The second parameter of base-heading-sizes-calc() function must be a string, #{type-of($font-family)} was given.
